# Slow Glow — установка на домашний экран (PWA)

В `App.jsx` уже встроена авто-настройка PWA: при запуске приложение само добавляет
манифест, иконку, тему и meta-теги. Этого достаточно, чтобы:

- **iPhone / iPad (Safari):** «Поделиться» → **«На экран Домой»** — приложение
  откроется как отдельное, в полный экран, со своей иконкой ✦ (без панели браузера).
- **Android (Chrome):** в меню браузера появится **«Установить приложение» /
  «Добавить на главный экран»**.

## Для самой надёжной установки и работы офлайн (необязательно)

Если ты выкладываешь приложение на свой сайт (https), положи рядом с `index.html`
эти файлы из архива и добавь 3 строки в `<head>`:

```
icon-192.png
icon-512.png
manifest.webmanifest
service-worker.js
```

В `<head>` страницы:

```html
<link rel="manifest" href="manifest.webmanifest" />
<meta name="theme-color" content="#FAF8F1" />
<link rel="apple-touch-icon" href="icon-192.png" />
```

И один раз зарегистрируй service worker (в любом своём JS или перед `</body>`):

```html
<script>
  if ("serviceWorker" in navigator) {
    navigator.serviceWorker.register("service-worker.js").catch(function(){});
  }
</script>
```

Готово — на Android появится полноценная установка, а приложение будет
открываться даже без интернета.

## Важно про фотографии
Большинство изображений в приложении указывают на папку `/img/...`. Где эта папка
есть (как на твоём хостинге) — показываются фото. Где её нет — Slow Glow аккуратно
рисует фирменную плашку-градиент с иконкой (никаких пустых белых рамок).
